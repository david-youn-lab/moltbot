# 테스트

통합 테스트 모듈

## 테스트 구조

```
테스트/
├── unit/           # 단위 테스트
├── integration/    # 통합 테스트
├── e2e/            # 종단간 테스트
└── fixtures/       # 테스트 데이터
```

## 실행 방법

```bash
# 전체 테스트
pytest

# 특정 모듈
pytest 테스트/unit/test_command_parser.py

# 커버리지 포함
pytest --cov=. --cov-report=html
```

## 테스트 대상

### 단위 테스트
- 명령어 파서
- 음성 인식기
- MQTT 클라이언트
- 설정 관리

### 통합 테스트
- 컨트롤타워 API
- 기기 제어 흐름
- MQTT 메시지 전달

### 종단간 테스트
- 음성 → 텍스트 → 명령 → 기기 제어
- 전체 시나리오 테스트
